DT 770 Headphone Compensation
All presets are calibrated for the DT 770 Pro 80Ω. Treble values are rolled back to compensate for the 8–10kHz brightness peak. Apply a Global EQ high-frequency cut via Menu → Global EQ as a universal correction across all presets. Cabinet models use 2x10 Redhead where possible for balanced headphone response. Reverb levels are set slightly higher than speaker use to compensate for the dry, in-head feel of closed-back headphones.
Pork & Pickle — Ground Rules
🥒
Signal Chain Rules
Combined presets remove amp OD/fuzz when P&P is active — no gain stacking.  ·  P&P Drive is set 1–2 points lower in Combined tabs vs. P&P-only tabs.  ·  Flea Funk Combined removes the Envelope Filter — it conflicts with the fuzz circuit.  ·  Jazz MIM must run in Passive Mode into any fuzz/OD circuit (exceptions noted per preset).  ·  All presets calibrated for DT 770 Pro 80Ω headphones.
